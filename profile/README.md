# 拍手游戏(clappinggame)
拍手游戏(clappinggame)是一种即时战略性心理博弈游戏，是只用手就能玩的游戏。它用手势来传达信息，对战方便，可以利用很多碎片化的时间，即开即玩，游戏时间短，甚至可以很多人一起玩甚至远程玩！
